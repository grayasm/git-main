debian-kernel

Make sure backports repo is added to /etc/apt/source.list

apt-cache search linux-image
apt-cache search linux-source

To install from backports

apt-get -t squeeze-backports install "package"

To find current kernel parameters

cat /boot/config-`uname -r` | grep HZ
cat /boot/config-`uname -r` | grep PREEMPT

Is timer frequency disabled ?

CONFIG_NO_HZ=y   #disables the next
CONFIG_HZ=250    #parameter

High resolution timers take priority.

cat /boot/config-`uname -r` | grep CONFIG_HIGH_RES_TIMERS
CONFIG_HIGH_RES_TIMERS=y

News: 05 Apr. 2013

· Tragerile artileriei - Obuzier cal. 152 mm model 1981
  Conducerea focului Divizionului (Batalionului) de Artilerie cal. 152mm

  uxcfart - 0.1b (beta)
  Programul a fost creat pentru instruirea specialistilor Divizionului de Artilerie cal. 152mm
  in calculul elementelor la obiective prin pregatire completa si transport de foc.

  Codul sursa poate fi descarcat de la: ftp://vasilian.dyndns.org/uxcfart-0.1b/uxcfart.tar.gz
 
  Pentru Linux/BSD/MacOS/*nix:
 
  mkdir -pv ~/Downloads
  cd ~/Downloads
  wget ftp://vasilian.dyndns.org/uxcfart-0.1b/uxcfart.tar.gz
  tar -zxvf uxcfart.tar.gz
  cd uxcfart/lib/misc
  make misc
  cd ../../app/uxcfart
  make all
 
  # run application
  cd src
  ./uxcfart
 
Pentru Windows:
  Descarcati executabilul de la: ftp://vasilian.dyndns.org/uxcfart-0.1b/uxcfart.zip
  Have fun!

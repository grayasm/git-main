Summary

An exception class

Synopsis

 
#include <stdexcept>

Description

Objects of class out_of_range are thrown when a parameter is outside the valid range. The class is derived from logic_error, and thrown by bitset, deque, string, vector, and others.

Interface

 
    namespace std {
  class out_of_range : public logic_error {
  public:
    explicit out_of_range(const string&  what_arg);
  };
}

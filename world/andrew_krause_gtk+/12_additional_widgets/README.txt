Chapter 12 - Additional GTK+ Widgets

        These are a number of widgets that did not fit into previous chapters.
        Here are described the following:
        - how to use the drawing widgets (GtkDrawingArea, GtkLayout)
        - how to use GtkCalendar to get info about months of the year
        - how to use GTK+2.10 for recent files, printing and status icons
        - how to have automatic completion with GtkEntryCompletion

Table 1-2 Different types of shell to declare

#!/bin/sh                - declares a Bourne shell
#!/usr/bin/sh            - declares a Bourne shell

#!/bin/ksh               - declares a Korn shell
#!/usr/bin/ksh           - declares a Korn shell

#!/bin/csh               - declares a C shell
#!/usr/bin/csh           - declares a C shell

#!/bin/bash              - declares a Bourne-Again (Bash) shell
#!/usr/bin/bash          - declares a Bourne-Again (Bash) shell
